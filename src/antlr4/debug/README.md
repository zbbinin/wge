# Debugging with predefined scripts
- [debug_parser.sh](debug_parser.sh) is a shell script that print a parse tree for the input file `input.txt`.
- [debug_lexer.sh](debug_lexer.sh) is a shell script that prints a lexer tree for the input file `input.txt`.
- [input.txt](input.txt) is a simple text file that contains the input to be parsed. We can modify this file to test different inputs.
